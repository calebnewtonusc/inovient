# Inovient: AI SaaS Pricing and Market Strategy

A multi-page strategy site built by Kappa Theta Pi (USC) for Inovient, delivering market research, competitor analysis, ROI modeling, pricing strategy, and product positioning for Inovient's Morpheus AI platform.

**Live:** [inovient.vercel.app](https://inovient.vercel.app)
**Client:** [inovient.io](https://www.inovient.io)

## Pages

| Route | Content |
|-------|---------|
| `/` | Executive overview: market size, pricing summary, key metrics |
| `/market-research` | TAM/SAM/SOM breakdown, market trends, growth projections |
| `/competitor-analysis` | Competitor landscape, feature matrix, positioning gaps |
| `/roi-modeling` | ROI projections, cost-benefit analysis, payback period |
| `/pricing-strategy` | Tiered pricing model, feature tiers, GTM recommendations |
| `/product-positioning` | Positioning framework, ICP definition, messaging architecture |

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Fonts**: Geist
- **Charts**: Recharts
- **Icons**: Lucide React
- **Brand Color**: #22D3A4 (teal green)
- **Deployment**: Vercel

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
